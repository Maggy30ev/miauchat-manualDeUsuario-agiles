Preguntas Frecuentes

¿Que es Miuchat?

Es una aplicación de chat web diseñada para escritorio que te permite mantenerte conectado con otros usuarios a través de salas de chat. Puedes invitar a otros usuarios a unirse a las salas simplemente compartiendo un código, lo que hace que sea más rápido y conveniente comenzar a comunicarte. Además, no es necesario agregar contactos, ya que cualquier persona con una cuenta en la aplicación puede unirse a las salas de chat. Esta función de acceso mediante códigos diferencia nuestra aplicación de otras aplicaciones de chat existentes.

¿Como crear una cuenta?

Para generar una cuenta solo necesitas ingresar a la página de registro, rellenar el formulario con tú datos y en cuestión de segundos podrás disfrutar de los beneficios que te da esta aplicación. Para más detalles, visita la sección Sesión y en el apartado Crear cuenta podrás ver el paso a paso para poder registrarte.

Se pueden dar tres casos de error al momento de registrarse:

  1. Ingresemos un usuario que ya existe en el sistema.
  2. No seleccionar una foto de perfil
  3. Las contraseñas no coincidan

Salas de chat

Crear una sala de chat

Para crear una sala de chat primero debemos ingresar con nuestro usuario y contraseña correspondientes. Una vez dentro de la aplicación realizamos los siguientes pasos, donde el número del paso corresponde con los circulos marcados en cada una de las imagenes:

  1. Seleccionamos el botón de añadir, ubicado en la parte inferior derecha, detro de la sección de mensajes.
  2. Se muestra una ventana emergente. Donde dentro del campo de texto debemos ingresar el código de la sala.
  3. Presionar el botón para crear la sala.
  4. Se mostrará un mensaje de confirmación en la esquina inferior derecha de la pantalla.
  5. Se mostrará la nueva sala creada en la sección de salas de chat.
  6. Al dar click en la sala esta se mostrará del lado derecho de la pantalla y podremos visualizar el código de acceso a la sala.

Unirse a una sala de chat

Para unirse a una sala de chat primero debemos ingresar con nuestro usuario y contraseña correspondientes. Una vez dentro de la aplicación realizamos los siguientes pasos, donde el número del paso corresponde con los circulos marcados en cada una de las imagenes:

  1. Seleccionamos el botón de añadir, ubicado en la parte inferior derecha, detro de la sección de mensajes.
  2. Se muestra una ventana donde debemos presionar el enlace para mostrar el formulario de ingreso a una sala.
  3. Se muestra una ventana emergente. Donde dentro del campo de texto debemos ingresar el código de la sala.
  4. Presionar el botón para crear la sala.

Abandonar una sala de chat

Para abandonar una sala de chat primero debemos ingresar con nuestro usuario y contraseña correspondientes. Una vez dentro de la aplicación realizamos los siguientes pasos, donde el número del paso corresponde con los circulos marcados en cada una de las imagenes:

  1. Dentro de la pantalla debemos presionar sobre el chat para abrir la interfaz de los mensajes.
  2. Dentro de la ventana en la parte superior derecha, debemos seleccionar el botón de opciones.
  3. Una vez desplegado el menú de opciones, seleccionamos la opción de Abandonar Sala.
  4. Se mostrará una ventana donde podrás elegir a un nuevo usuario como administrador de la sala
  5. Y luego deberas presionar el botón de Abandonar para aplicar los cambios.

  4. Se mostrara una ventana de emergencia, seleccionar el botón de Eliminar para dar de baja el chat en la aplicación.

Generales

Descripción

MiauChat es un proyecto realizado para la asignatura de Desarrollo Ágil de Software impartida en la Universidad Tecnológica de la Mixteca. El proyecto utiliza la metodología ágil de desarrollo Extreme Programming (XP). MiauChat consiste en un chat que permite a los usuarios conectarse a un mediante un login en la plataforma para poder crear salas de chat y poder comunicarse entre ellos.

Instalación y Ejecución

Para la instalación de la aplicación en su sistema operativo previamente es necesario contar con los siguientes requerimientos de software.

Requerimientos

El proyecto está compuesto por dos partes, el frontend y el backend. El frontend está desarrollado con React y el backend con Node.js.

  • Versión de Node.js mayor a igual a la v18.0.0.
  • Sistema de gestión de paquetes npm.
  • Sistema gestor de bases de datos MySql.

Instalación

La aplicación, al estar basada en la web, tiene un modelo de trabajo cliente-servidor, el código fuente está disponible desde GitHub. Para acceder al código fuente, sigua los siguientes pasos:

  1. Clonar los repositorios:

    • Frontend: miauchat-frontend-agiles.
    • Backend: miauchat-backend-agiles.

  2. Mediante una terminal acceder a la raíz de las carpetas, deberá ejecutar el siguiente comando para instalar las dependencias necesarias: npm install.

Ejecución

Pasos para la ejecución de los scripts encargados de la compilación del servidor y cliente de la aplicación.

  1. En una terminal, dentro del proyecto miauchat-backend-agiles ejecutar el siguiente comando: npm run start:dev.
  2. De igual manera, dentro del proyecto miauchat-frontend-agiles ejecutar el siguiente comando: npm run dev.
